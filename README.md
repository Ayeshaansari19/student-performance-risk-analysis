## Student Performance Risk Analysis

### Objective
Identify students at risk of failing using academic and behavioral data to support early intervention.

### Tools Used
- Python (Pandas, NumPy, Scikit-learn)
- Google Colab

### Approach
- Exploratory Data Analysis to identify key risk factors
- Logistic Regression model for pass/fail prediction
- Evaluation using recall and confusion matrix

### Key Result
- Achieved **87% recall** in identifying at-risk students
# student-performance-risk-analysis
Mini data science project to identify at-risk students using logistic regression.
